# Causal Effect of Naturalization on Immigrant Wages
## STAT 209 Final Project
This repository contains code for the paper “The Causal Effect of Naturalization on Immigrant Wages in the United States.”

The analysis uses observational methods, including Outcome Regression, Propensity Score Matching, and Augmented Inverse Probability Weighting (AIPW), to estimate the wage premium associated with naturalization for U.S. immigrants.

This repository **does not** include any microdata from the American Community Survey (ACS).
Per the IPUMS USA data-use agreement, users must request and download their own extract from: https://usa.ipums.org/usa/
